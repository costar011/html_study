html이란? 웹페이지 콘텐츠 안의 꺽쇠 괄호에 둘러싸인 '태그' 로 되어있는 html 요소 형태로 작성 태그란? html를 구성하기 위한 요소 및 꺽쇠 괄호에 둘러 쌓여 있음 : <태그명> 문단의 구분 <p> 태그 : 한 단락을 구분하는 태그로 한 문장을 끝내고 다음 문장을 시작할 때 한 줄을 띈 후에 시작하고자 할 때 사용하는 태그 <br> 태그 : 줄 바꿈 태그로 p태그 명령과 다른 한 점은 문장을 끝내고 다음 문장을 시작할 때 줄만 바꾸는 태그 <pre> 태그 : 입력된 문자들은 어떤 문자라도 입력할 떄의 내용 그대로 출력되는 태그 <nobr> 태그 : 웹 브라우저의 크기가 달라져도 줄을 바꾸지 못하게 하는 태그 <div> 태그 : p태그와 마찬가지로 문단을 구별하는 태그로 p태그와 다른 점은 div태그는 p태그로 나뉘어진 문단을 하나의 묶음 형태로 정렬할 수 있고 p태그는 사용한 위치에 한줄의 공백이 생기지만 div태그는 공백이 생기지 않는 태그 <hr> 태그 : 웹브라우저에 수평선을 만들어주고 속성에 따라 수평선의 색상과 굵기, 길이 등을 다양하게 지정할 수 있다.
